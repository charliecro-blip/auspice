# Auspice Prototype

This folder is reserved for the first clickable Auspice prototype.

## Prototype Goal

Build a static single-page React prototype that proves the core experience:

> I open Auspice and immediately understand what kind of time I am in, what fits now, and how the week wants to move.

## Initial Screens

1. **Today**
   - Current time weather
   - Planetary hour
   - Moon condition
   - Local angularity highlight
   - Current recommendation
   - What Now assistant card

2. **Week**
   - Week tone
   - Seven-day rhythm map
   - Suggested sessions
   - Not This Week section

3. **Tasks**
   - Manual sample task list
   - Task categories
   - Fit-now / better-later / incubating groupings

## Prototype Constraints

Do not build yet:

- real ephemeris calculations
- Google Calendar integration
- Notion integration
- user accounts
- database persistence
- calendar write-back
- full electional chart calculation

Use mock astrology data and the seed rules in `../rules/auspice-seed-rules-v1.json`.

## Success Criteria

The prototype succeeds if it feels:

- spacious
- useful
- non-coercive
- rhythm-aware
- astrologically meaningful
- more like a living almanac than a productivity dashboard
