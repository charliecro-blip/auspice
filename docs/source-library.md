# Auspice Source Library

## Purpose

This document tracks the source material used to build the Auspice rule logic.

The goal is not to gather infinite material. The goal is to collect enough good source material to build a practical, source-informed rulebook for the app.

## Source Categories

### 1. Electional Astrology - Core

Highest priority sources for timing logic.

- Vivian Robson, *Electional Astrology*
- Astrology Podcast Ep. 190: Electional Astrology with Chris Brennan and Leisa Schaim
- Astrology Podcast Ep. 292: Defining the Void-of-Course Moon
- Seven Stars Astrology: Dorotheus on Asking Favors
- Mary Fortier Shea, *Electional Astrology* if obtained later
- Christopher Warnock, *Renaissance Astrology Electional Compilation* if obtained later
- Guido Bonatti, *On Elections* if obtained later

### 2. Traditional Background

### 3. Daily Timing and Rhythm

## Extraction Template

For each source, extract:

- Source title
- Author
- Tradition/context
- Source type
- Key concepts
- Electional rules
- Timing factors
- Event categories
- Task correspondences
- User-facing language
- App logic implications
- Notes on what to modernize or soften
- Confidence level
